# Chob.Shop - Affiliate E-Commerce Platform

A simple and dynamic interface for showcasing Shopee affiliate products.

## Features

- **Automated Social Media Posting**: Posts to Facebook, X (Twitter), and Threads.
- **AI-Powered Captions**: Generates engaging review-style captions using Gemini AI.
- **CSV Import**: Easily import Shopee products via CSV.
- **Modern UI**: Clean and responsive design.

## Setup

1. Clone the repository.
2. Run `npm install`.
3. Fill in your `.env` variables (see `.env.example`).
4. Run `npm start`.

## Deployment

Refer to `deploy_guide.md` for instructions on deploying to cPanel / DirectAdmin.
